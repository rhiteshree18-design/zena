ℹ️ About

ZENA (Zero-touch ENhanced Assistant) is an AI-powered gesture recognition system built using MediaPipe Hands and JavaScript.

It demonstrates real-time hand tracking, gesture-based control, and human-computer interaction through intuitive motion commands.

Developed as part of an academic project to explore touchless accessibility and natural user interfaces.
